Overview 
Reachable full stack project:
Reachable is a full-stack web platform designed to help people with disabilities discover accessible places and services in Egypt. The platform categorizes locations by type, allowing users to find hospitals, malls, restaurants, schools, public transport, adaptive sports, job opportunities, and medical supply shops that accommodate their needs.

Problem Statement:
People with disabilities in Egypt often struggle to find reliable information about whether public places are accessible. Reachable solves this by providing a centralized, categorized directory of verified accessible locations — making it easier to navigate daily life.

Features:
•	8 Category Sections — Hospitals, Restaurants, Schools, Malls, Public Transport, Adaptive Sports, Jobs, Supplies
•	Authentication System — User registration and login via Laravel Auth
•	Feedback & Rating System — Users can leave comments and rate schools (1–5 stars) with validation
•	Job Filtering by Disability Type — Users select their disability type (mobility, hand, speech, vision) to view relevant job listings
•	Video Hero Banner — Autoplay muted background video on the homepage
•	Scroll Animations — Smooth AOS-powered entrance animations on all category cards
•	Admin Panel — Backend admin area for managing categories and content
